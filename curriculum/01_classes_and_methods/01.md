# Bevezetés a C++ Objektumorientált Programozásba (OOP)

Az OOP az **Object-Oriented Programming** (Objektumorientált programozás) rövidítése.

### Procedurális vs. Objektumorientált programozás

* **Procedurális programozás:** Olyan eljárások (procedures) vagy függvények írásáról szól, amelyek műveleteket hajtanak végre az adatokon.
* **Objektumorientált programozás:** Olyan objektumok létrehozásáról szól, amelyek egyszerre tartalmaznak adatokat és a hozzájuk tartozó függvényeket is.

### Miért érdemes az OOP-t használni? 💡

Az objektumorientált programozásnak számos előnye van a hagyományos procedurális módszerekkel szemben:

1. **Gyorsabb és könnyebben végrehajtható:** Az OOP struktúrája hatékonyabb kódolást tesz lehetővé.
2. **Tiszta struktúra:** Átláthatóbbá teszi a program felépítését.
3. **DRY elv (Don't Repeat Yourself - Ne ismételd önmagad):** Segít a C++ kód karbantartásában, módosításában és hibakeresésében, mivel csökkenti a kódismétlést.
4. **Újrafelhasználhatóság:** Kevesebb kóddal és rövidebb fejlesztési idővel hozhatunk létre teljesen újrafelhasználható alkalmazásokat.

### Osztályok (Classes) és Objektumok (Objects) 🏗️

Az OOP két legfontosabb fogalma az **Osztály** és az **Objektum**.

A legegyszerűbb úgy elképzelni, hogy az **Osztály** a tervrajz (sablon), az **Objektum** pedig a tervrajz alapján elkészült konkrét példány.

**Példa:**

| Osztály (Class) | Objektumok (Objects) |
| --- | --- |
| `Fruit` (Gyümölcs) | `Apple` (Alma) |
|  | `Banana` (Banán) |
|  | `Mango` (Mangó) |

| Osztály (Class) | Objektumok (Objects) |
| --- | --- |
| `Car` (Autó) | `Volvo` |
|  | `Audi` |
|  | `Toyota` |

***Tehát:*** az osztály (Class) egy sablon az objektumok létrehozásához, az objektum (Object) pedig az osztály egy példánya. Amikor létrejön egy objektum, az örökli az osztályban meghatározott összes változót és függvényt.

# Osztályok és Objektumok (Classes/Objects)

C++-ban az osztályokat a `class` kulcsszóval hozzuk létre.

### Osztály létrehozása

Hozzon létre egy `MyClass` nevű osztályt két változóval (ezeket **attribútumoknak** nevezzük): egy `myNum` nevű egész számmal (`int`) és egy `myString` nevű szöveggel (`string`).

```cpp
class MyClass {       // Az osztály neve
  public:             // Hozzáférési specifikáció (erről később részletesebben)
    int myNum;        // Attribútum (egész szám)
    string myString;  // Attribútum (szöveg)
};

```

⚠️ **Fontos:** Ne felejtsd el a pontosvesszőt `;` az osztály definíciójának legvégén!

### Objektum létrehozása

Miután definiáltuk az osztályt (a tervrajzot), elkezdhetjük használni azt objektumok létrehozására. Ezt általában a `main()` függvényben tesszük, hasonlóan ahhoz, ahogy egy egyszerű változót deklarálunk (pl. `int x;`).

```cpp
int main() {
  MyClass myObj;  // Objektum létrehozása a MyClass-ból
  return 0;
}

```

### Attribútumok elérése

Az osztály attribútumait (változóit) a pont `.` szintaxissal érhetjük el az objektumon keresztül.

**Teljes példa:**

```cpp
#include <iostream>
#include <string>
using namespace std;

// Osztály definíciója
class MyClass {
  public:
    int myNum;
    string myString;
};

int main() {
  MyClass myObj;  // Objektum létrehozása

  // Attribútumok elérése és értékadás
  myObj.myNum = 15;
  myObj.myString = "Some text";

  // Értékek kiírása
  cout << myObj.myNum << "\n";
  cout << myObj.myString;
  return 0;
}

```

# Osztály Metódusok (Class Methods)

A metódusok lényegében **függvények**, amelyek az osztályhoz tartoznak. Kétféleképpen definiálhatjuk őket:

1. Az osztály definícióján **belül**.
2. Az osztály definícióján **kívül**.

### 1. Definíció az osztályon belül

Ez a legegyszerűbb módszer. A függvényt közvetlenül az osztályon belül írjuk meg.

```cpp
class MyClass {
  public:
    void myMethod() {  // Metódus definiálása az osztályon belül
      cout << "Hello World!";
    }
};

int main() {
  MyClass myObj;     // Objektum létrehozása
  myObj.myMethod();  // Metódus meghívása
  return 0;
}

```

### 2. Definíció az osztályon kívül

Gyakran jobb, ha szétválasztjuk a deklarációt és a megvalósítást (ez átláthatóbbá teszi a kódot). Ilyenkor az osztályon belül csak deklaráljuk a metódust, és kívül írjuk meg a működését.

Ehhez használnunk kell az osztály nevét és a `::` operátort (ezt **scope resolution operator**-nak, vagy hatókör feloldó operátornak hívják).

```cpp
class MyClass {
  public:
    void myMethod();   // Csak deklaráljuk a függvényt
};

// Itt definiáljuk az osztályon kívül
void MyClass::myMethod() {
  cout << "Hello World!";
}

int main() {
  MyClass myObj;
  myObj.myMethod();
  return 0;
}

```

### Paraméterek

A metódusok ugyanúgy fogadhatnak paramétereket, mint a hagyományos függvények.

```cpp
class Car {
  public:
    int speed(int maxSpeed);
};

int Car::speed(int maxSpeed) {
  return maxSpeed;
}

```

# Feladatok

### 1. Az Alapok: A "Kutya" osztály 🐕

**Feladat:** Hozz létre egy `Kutya` nevű osztályt (`class`). Egyelőre ne legyen benne semmi (üres osztály). A `main()` függvényben hozz létre egy `bodri` nevű objektumot ebből az osztályból.

### 2. Adatok tárolása 📝

**Feladat:** Egészítsd ki a `Kutya` osztályt két publikus attribútummal:

* `nev` (szöveg / string)
* `kor` (egész szám / int)
A `main()`-ben adj értéket a `bodri` objektum adatainak (pl. "Bodri", 3), majd írasd ki őket a konzolra.

### 3. Viselkedés hozzáadása (Metódus belül) 🗣️

**Feladat:** Írj egy `ugat()` nevű metódust a `Kutya` osztályon **belül**. A metódus írja ki a konzolra, hogy `"Vau-vau!"`. Hívd meg ezt a metódust a `bodri` objektumon.

### 4. Metódusok szervezése (Metódus kívül) 🎂

**Feladat:** Deklarálj egy `szuletesnap()` nevű metódust az osztályon belül, de a kifejtését (definícióját) az osztályon **kívül** végezd el (ne felejtsd el a `::` operátort!).
Ez a metódus növelje meg a kutya korát 1-gyel, és írja ki: `"Boldog szülinapot! Most már [kor] éves vagy."`

### 5. A Végső Kihívás: Téglalap Kalkulátor 📐

**Feladat:** Készíts egy teljesen új `Teglalap` osztályt.

* Legyen két attribútuma: `szelesseg` és `magassag` (egész számok).
* Legyen egy `teruletSzamitas()` metódusa, ami **visszaadja** (return) a téglalap területét (szélesség * magasság).
* A `main()`-ben hozz létre két különböző téglalap objektumot más-más méretekkel, és írasd ki mindkettő területét.
