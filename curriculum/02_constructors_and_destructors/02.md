# Konstruktorok (Constructors)

A konstruktor egy speciális metódus (függvény), amely **automatikusan lefut**, amikor létrehozunk egy objektumot az osztályból.

Fő célja általában az, hogy kezdőértéket adjon a változóknak (attribútumoknak).

**A konstruktor szabályai:**

1. A neve **teljesen megegyezik** az osztály nevével.
2. **Nincs** visszatérési típusa (még `void` sem!).

### Példa konstruktorra

Itt egy példa, ahol a konstruktor automatikusan kiír egy üzenetet létrehozáskor:

```cpp
class MyClass {     // Osztály
  public:           // Hozzáférési specifikáció
    MyClass() {     // Konstruktor (neve egyezik az osztállyal)
      cout << "Hello World!";
    }
};

int main() {
  MyClass myObj;    // Objektum létrehozása (ez meghívja a konstruktort!)
  return 0;
}

```

### Konstruktor paraméterekkel

A konstruktoroknak adhatunk paramétereket is, így létrehozáskor azonnal beállíthatjuk a kezdőértékeket.

```cpp
class Car {
  public:
    string brand;
    string model;
    int year;
    
    // Konstruktor paraméterekkel
    Car(string x, string y, int z) {
      brand = x;
      model = y;
      year = z;
    }
};

int main() {
  // Objektum létrehozása és adatok megadása egy lépésben
  Car carObj1("BMW", "X5", 1999);
  Car carObj2("Ford", "Mustang", 1969);

  cout << carObj1.brand << " " << carObj1.model << " " << carObj1.year << "\n";
  cout << carObj2.brand << " " << carObj2.model << " " << carObj2.year << "\n";
  return 0;
}

```

### Az inicializáló lista (Member Initializer List)

A C++ modern és hatékonyabb módszert is kínál a tagváltozók beállítására: ez az **inicializáló lista**. Ezt a konstruktor fejléce után, egy kettőspont (`:`) bevezetése után használjuk.

**Miért jobb ez?**

* **Hatékonyság:** A változók rögtön a létrehozásuk pillanatában megkapják az értéküket, nem pedig utólagos értékadással a függvénytörzsben.
* **Kötelező használat:** Vannak esetek (pl. `const` változók vagy referenciák az osztályban), amikor csak így adhatunk értéket.

**Példa az inicializáló lista használatára:**

```cpp
class Car {
  public:
    string brand;
    string model;
    int year;

    // Inicializáló lista használata (hatékonyabb!)
    Car(string x, string y, int z) : brand(x), model(y), year(z) {
        // Itt a függvénytörzs akár üres is maradhat
    }
};

```

### Konstruktor Túlterhelés (Overloading)

Ahogy a normál függvényeknél, úgy a konstruktoroknál is létezik a **túlterhelés**. Ez azt jelenti, hogy egy osztálynak lehet **több konstruktora** is, amennyiben a paramétereik száma vagy típusa eltér.

Így létrehozhatsz objektumot adatok nélkül (üresen), vagy adatokkal is.

---

**1. Gyakorló feladat:**

Térjünk vissza a saját `Car` osztályodhoz.

1. Írj hozzá egy konstruktort, amely 3 paramétert vár (`brand`, `model`, `year`), és beállítja az osztály változóit ezekre az értékekre.
2. A `main` függvényben hozz létre egy `carObj1` nevű autót, és a létrehozás pillanatában add meg neki az adatait (pl. "Toyota", "Corolla", 2010).
3. Írasd ki az autó adatait.

# Destruktorok (Destructors)

A destruktor egy olyan speciális metódus, amely akkor fut le automatikusan, amikor az objektum **megszűnik** (például amikor a program kilép a függvényből, vagy véget ér a `main` blokkja).

Míg a konstruktor előkészíti a terepet (memóriát foglal, értékeket ad), a destruktor **takarít** maga után (memóriát szabadít fel, fájlokat zár be).

**Szabályok:**

1. A neve megegyezik az osztály nevével, de egy hullámvonal (`~`) van előtte.
2. Nem lehet paramétere.
3. Nem lehet visszatérési értéke.
4. Osztályonként csak **egy** lehet belőle.

### Példa destruktorra

Itt egy angol nyelvű példa, ahol láthatod, hogyan működik a születés és a törlés sorrendje:

```cpp
#include <iostream>
using namespace std;

class Hero {
  public:
    // Constructor
    Hero() {
      cout << "Hero created!" << endl;
    }

    // Destructor
    ~Hero() {
      cout << "Hero defeated... Game Over." << endl;
    }
};

int main() {
  Hero myHero; // Constructor is called here: "Hero created!"
  
  cout << "The hero is fighting..." << endl;

  return 0; 
  // Destructor is called automatically here: 
  // "Hero defeated... Game Over."
}

```

---

**2. Gyakorló feladat:**

Egészítsd ki a meglévő `Car` osztályodat egy **destruktorral**!

1. A destruktor neve legyen `~Car`.
2. A törzsében írass ki egy búcsúüzenetet, ami tartalmazza a kocsi nevét is. Példa üzenet: *"Goodbye, [nev]."*
3. Futtasd le a programot, és figyeld meg, hogy a `main()` függvény végén (a `return 0;` után) megjelenik-e az üzenet.
